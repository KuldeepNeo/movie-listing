# Project Boundaries

## Included

* React Application
* Use "https://jsonfakery.com/movies/paginated" api to fetch the Movie Details
* Use "https://www.netflix.com/in/" Netflix Application design to design the Movie listing and Movie details page
* Responsive Design
* Fallback Handling

---

## Constraints

### Pages

Single Page for Movie listing.
Single Page for Movie details page.

### Navigation

- On click of movie card navigate to movie details page.
- User can navigate back to the movie listing page using the browser's back button.

### State Management

React State only.
